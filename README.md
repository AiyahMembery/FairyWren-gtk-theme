## FairyWren-gtk-theme

FairyWren is a flat Material Design theme for GTK 3, GTK 2 and Gnome-Shell which supports GTK 3 and GTK 2 based desktop environments like Gnome, Budgie, etc.

This theme is based on Layan gtk theme of nana-4. thanks heaps to the vinceliuce for the theme.

#### Layan gtk theme: https://github.com/vinceliuice/Layan-gtk-theme
